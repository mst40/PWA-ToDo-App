# ToDo web app

**https://mst40.github.io/PWA-ToDo-App/**

This project is a Progressive Web App (PWA) for managing todos. It leverages modern web technologies and libraries.

## Dependencies

- @emotion/react
- @emotion/styled
- @mui/material
- localforage
- react
- react-dom
- react-qrcode-logo
